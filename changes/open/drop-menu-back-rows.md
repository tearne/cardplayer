# Drop the Back rows from menus

## Intent

Several menus carry an explicit "Back" row as their last entry — Leveling, Alarms, Alarm Editor, and Set Current Time. Now that Esc (`` ` ``) backs out of every screen, and Del also backs out of menus, the Back row is redundant navigation. Drop it from all of them so the menus are shorter and the back-out model is the single standardised Esc, applied consistently across the app.

Spun off from `leveller-distortion`, where the same redundancy showed up in the Leveling sub-menu — kept there for now so all the menus lose Back together rather than piecemeal.
